# 現場点検PWA (Inspection Support Tool)

現場作業者の「使いやすさ」に特化した、インストール可能なWebアプリです。
片手での操作を考慮した **1問1答・自動進捗UI** により、点検作業を高速化します。

## ✨ このアプリでできること

1.  **日付に縛られない柔軟な検索**
    * 日付を指定せずに場所一覧を表示し、現場に到着してから「今日」の点検として即座に入力開始できます。
2.  **迷わない地図機能**
    * 地図画面には余計な検索窓を置かず、一覧で選んだ場所をスマートに表示。
    * マーカーをクリックして「ナビ」ボタンを押せば、**Googleマップが起動**して現場まで案内します。
3.  **止まらない入力体験（1問1答UI）**
    * OK/NGボタンやリストを選んだ瞬間に、**次の項目へ自動でスクロール**。
    * 入力中の項目がハイライトされ、それ以外がグレーアウトするため、集中して入力できます。
4.  **オフライン対応**
    * PWA（Progressive Web App）としてホーム画面に追加可能。
    * サービスワーカーにより、電波の入りにくい地下や山間部でも動作します。

## 🚀 導入手順

### 1. ファイルを配置
サーバー（GitHub Pages等）に以下のファイルをアップロードしてください。
* `index.html`
* `styles.css`
* `app.js`
* `sw.js`
* `manifest.webmanifest`
* `icon-192.png` / `icon-512.png`

### 2. インストール
* **Android (Chrome):** メニューから「アプリをインストール」を選択。
* **iOS (Safari):** 共有ボタンから「ホーム画面に追加」を選択。

### 3. データの準備
CSV形式のマスターデータと場所データを用意し、「設定」「取込」タブから取り込みます。
